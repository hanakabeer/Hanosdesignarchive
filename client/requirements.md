## Packages
framer-motion | Complex scroll-triggered animations and layout transitions
clsx | Conditional class names for cleaner component logic
tailwind-merge | Merging tailwind classes effectively

## Notes
The application uses a high-end "Dark Atmospheric" theme.
Typography relies on 'Bricolage Grotesque' (Display) and 'Inter' (Body).
Components are isolated in `client/src/components/sections/` using CSS Modules where requested for specific stylization, mixed with Tailwind for layout.
Backend API at `/api/projects` provides content.
